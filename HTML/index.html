Wells & Alerts

Wells have borders and grey background. Use div class="well" to create.
Small wells have borders and grey background, but less padding. Use div class="well-sm" to create.
Alerts are like wells with color. Use div class="alert alert-success" for green theme.
Alerts are like wells with color. Use div class="alert alert-info" for blue theme.
Alerts are like wells with color. Use div class="alert alert-danger" for red theme.
×Alerts can be dismissed. Use a href="#" class="close" data-dismiss="alert" aria-label="close"
×Alerts can fade out when dismissed. Add "fade in" to the div class="alert"
